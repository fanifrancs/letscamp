# Let's Camp
A functional express app that lets you create and checkout interesting campgrounds

## 👀 Preview
Take a look right [here](https://lcamp.herokuapp.com)

## Technologies Used
- node
- express
- ejs
- mongodb
- passport
- all other packages in package.json

## Features
- authentication
- authorization
- create campgrounds
- comment
- delete campgrounds
